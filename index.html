使用指南

从配置到实战，六大板块覆盖外贸全场景

创建智能体

在 Accio Work 中按业务职能创建不同的智能体，每个专注于一个领域：市场洞察、供应商评估、专业技能定制、自我进化。

智能体喂养与训练

创建后可通过"知识喂养"提升专业度——上传行业报告、内部材料，让智能体学习你行业的独特逻辑。不喂养也可直接使用，Agent 已具备基础能力。

创建智能体团队

将多个智能体组合为"团队"，实现跨职能协作。例如将"竞品分析""选品""详情页优化"组合为完整运营团队。

设置定时任务

支持定时执行。例如"每天 9:00 自动分析近 7 天询盘并发送报告"，实现运营自动化。

CAD & 3D Render Generator 高阶版

专为游乐设备行业打造的 CID（产品效果图）设计与渲染技能，覆盖户外组合滑梯、攀岩墙、蹦床公园、秋千、摇摇乐、室内淘气堡等细分赛道，一键生成 CAD 技术图纸和 3D 渲染效果图。同时支持卫浴等品类的 CAD 与 3D 渲染需求。

触发词（中英文均可）：
中文：CAD图 / 3D渲染 / 效果图 / 技术图纸 / 工程图 / CID图
英文：CAD drawing / 3D render / rendering / visualization

如何开始？（指令）

为了能生成最精准的效果，通常需要提供以下信息：

  1. 产品类型（例如：户外森林风滑梯、室内海洋主题淘气堡、蹦床公园等）
  2. 核心参数（长宽高尺寸、主要材质、配色要求等）
  3. 目标场景（是放在户外草坪、沙地，还是室内商业空间？）
  4. 特殊需求（是否需要特定的品牌配色或特定的视角？）

使用示例

示例 1 — 直接说产品：
"帮我生成一套户外组合滑梯的 CAD 图纸和 3D 效果图"
→ Agent 会自动问你收集尺寸、材质、颜色等参数
示例 2 — 带参数说：
"室内蹦床公园，20m×15m，黑色钢架+荧光蓝绿橙配色，帮我生成全套图纸"
→ Agent 会直接开始生成
示例 3 — 指定用途：
"我需要一套攀岩墙的效果图用于投标方案"
→ Agent 会按投标文档风格生成（更正式、含合规标准）

输出内容（每次生成 6 张图）

CAD 技术图纸（3 张）：

  • CAD 俯视平面图（含尺寸标注 + 功能分区）
  • CAD 正立面图（含材质标注 + 层高标注）
  • CAD 节点详图（含连接方式 + 安全间距）

3D 渲染效果图（3 张）：

  • 3D 主视觉渲染（白底产品图）
  • 3D 场景效果图（实景安装效果）
  • 3D 多角度展示图（正面 / 背面 / 侧面 / 俯视）
迭代优化：如果某张图不满意，告诉 Agent"第 3 张细节图需要调整焊接标注"即可单独重新生成。

适用材质：镀锌钢管、LLDPE 工程塑料、304 不锈钢、实木、玻璃钢、PVC 软包——技能自动匹配英文 prompt 中的材质质感描述。

安全标准参考：EN1176（欧洲）、ASTM F1487（美国）、GB/T 27689（中国），根据目标市场自动选用。

安全提示：AI 生成的图纸为效果展示图，不可替代专业工程师的施工图纸。承重结构、安全间距等关键参数需以专业计算为准，实际生产前必须进行结构安全评估。

高转化 SOP NEW

专业的客户沉默唤醒 SOP，通过三阶画像 + 五维诊断 + 动态处方库，精准定位沉默原因并生成高打开率跟进邮件。

指令：您可以直接贴出一段聊天记录，或者回答以下 3 个基础问题让我开始分析：
  1. 客户类型：他是做主机厂配套、售后批发，还是终端维修？
  2. 沉默节点：他最后一次回你是聊到了什么？（报了价？确认了样件？还是聊到了运费？）
  3. 目前状态：沉默多久了？你中间追问过几次？用的是什么渠道？

技能具备能力：

  • 三阶客户画像 (A/B/C)：能根据关键词（如 IATF、Starter Order、VIN 码）瞬间锁定客户真实身份与核心痛点。
  • 五维诊断模型：从"客户、信息、渠道、信任、心理"五个深度维度切入，精准找出沉默的"病灶"。
  • 动态处方库：涵盖了从"需求确认"到"优雅退出"的 10 类精准中英文模板，每一条都能直接复制发送。
适用场景：客户报完价后沉默、样件确认后无下文、运费谈崩后失联、长期未回复的沉睡客户唤醒。

Agent 人格设置（最佳实践）

一、人格内核：直接给结论，不铺垫。禁止模糊表达，能判断必须判断。回答必须可落地。简洁是硬规则。
二、表达结构：结论 → 原因 → 可执行方案。不倒序，不绕圈。
三、执行协议：能自动完成的直接执行。发现错误立即修复。修改前先备份。
四、安全边界：对外发送/删除/修改重要数据必须二次确认。不暴露隐私信息。
五、核心信条：数据即真相。策略导向——将"发生了什么"转化为"我们该怎么做"。
