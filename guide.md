## 一、Accio Work 2.0 是什么

Accio Work 是为全球中小企业主和创业者量身定制的"生意大脑"。它不是简单的聊天机器人，而是一支会自主做生意的 Agent 团队。用户通过对话框与内置的多个专业 Agent 沟通并发出指令，Accio Work 就会开启 7x24 小时的自动工作模式。

**核心能力一览：**

- **全链路自动化**：7x24 小时自主完成市场分析、产品选品、寻找供应商、采购谈判、社媒推广的长流程任务。
- **30 分钟搭建网店**：给出创意，Accio 可在 30 分钟内从无到有搭建一个真实的 Shopify 独立站。
- **自带谈判专家属性**：安全调用邮箱权限，代替买家与供应商进行多轮邮件砍价，直到达成目标价格。
- **开放的 Skills 生态**：用户可以将独特的商业流程封装成"技能（Skills）"分享，能力库随全球用户使用不断扩充。
- **懂中国供应链**：依托阿里 26 年外贸数据，决策比通用模型更准确，幻觉更少。

**下载与注册：** 访问 https://www.accio-ai.com/login?sId=3zi8f11D ，使用邮箱注册登录即可。安装指导视频：https://weixin.qq.com/sph/A5Gpsdqfx4

## 二、基础配置：从零到一搭建你的 Agent

### 2.1 创建单个智能体

在 Accio Work 中，你可以按业务职能创建不同的智能体，每个智能体专注于一个领域。常见角色包括：

- **市场洞察**：实时分析海外需求，帮你看清全球风向，不再盲目选品。
- **供应商评估**：智能筛选优质货源，把控跨境供应链第一关。
- **专业技能定制**：将运营大牛的秘籍录入变成 AI 技能，让 Agent 24 小时照着干。
- **自我进化**：不只执行指令，还会复盘反馈，越用越懂你的生意。

### 2.2 智能体喂养与训练

创建智能体后，可以通过"知识喂养"提升其专业度。最有效的方式是上传行业报告、内部专业材料等，让智能体学习你所在行业的独特逻辑。当然也可以不喂养直接开干——Agent 已具备基础能力。

### 2.3 创建智能体团队

当单个智能体成熟后，可以将多个智能体组合为"团队"，实现跨职能协作。例如：将"竞品分析 Agent""选品 Agent""详情页优化 Agent"组合为一支完整的运营团队，由它们自动串联执行复杂任务。

### 2.4 设置定时任务

智能体支持定时执行任务。例如设置"每天早上 9:00 自动分析近 7 天询盘情况并发送报告"，实现运营自动化。

### 2.5 提示词指南

官方提供了提示词参考库，地址为：https://awbforzjbusiness.1d.alibaba-inc.com/#/prompts ，建议新手先从模板入手。

## 三、微信 / 企微绑定（消息渠道）

Accio Work 支持绑定微信（个人/企业微信/公众号），实现消息自动响应。

### 3.1 企业微信绑定（推荐，外贸/大客户场景最稳）

1. 打开 Accio Work → 左侧【消息渠道】→ 点击【企业微信/微信客服】
2. 用企业微信管理员扫码授权 → 选择要接入的企业/可见范围
3. 企微后台：应用管理 → 微信客服 → 开启并添加 Accio 机器人
4. 完成后客户微信/企微咨询 → Accio 自动 AI 接待/同步工作台

### 3.2 个人微信绑定（简易版，扫码即用）

1. Accio Work → 【配对授权/微信集成】→ 生成二维码
2. 手机微信扫码 → 确认授权（无需 AppID/Secret）
3. 绑定成功：微信发消息 → Accio 自动响应/同步处理

### 3.3 微信公众号绑定（官网/私域引流用）

1. 公众号后台 → 设置与开发 → 开发者 ID → 复制 AppID/AppSecret
2. Accio 【消息渠道】→ 公众号 → 粘贴 ID/密钥 → 提交
3. 配置 IP 白名单与接口 → 完成授权

**大客户场景关键要点：** 优先使用企业微信 + 微信客服，支持客户微信直接咨询、AI 自动回复、工单同步。管理员扫码授权即可，无需复杂开发。适用于外贸询盘、客户跟进、微信消息 AI 自动处理。

### 板块二：选品与产品策略

**适用场景：** 深度选品报告、蓝海机会扫描、产品组合取舍。

**技能包：** 选品 Skill（基础版 + 进阶版）

**标准指令模板：**
```
请基于「供应商选品分析」和「选品雷达」技能，为我分析以下供应商/品类：
[输入店铺网址或公司名称]

要求输出一份完整的深度选品报告，必须包含以下模块：
1. 供应商画像分析：提取年限、主营品类、响应率及核心认证
2. 产品线四象限分类：核心款、潜力款、走量款、不建议款
3. 跨平台需求验证：对比 Amazon、TikTok Shop 和 Google Trends 数据
4. 选品优先级矩阵：7 列详细表格
5. 重点产品深度拆解：1-2 个三星级产品的差异化机会
6. 1688 溯源推荐：5 家同类产业带头部工厂店
7. 全量参考链接补完
8. 3 步行动指南：今天、本周、本月具体建议
```

**进阶指令（高阶场景）：**

- **攻坚型**："针对 [类目/产品]，帮我拆解前 3 名头部竞品的 Review 痛点分布，并找出一个能作为我们新品卖点的'爆破口'。"
- **防御型**："分析 [竞争对手] 的流量结构和定价权重，评估如果他们降价 10%，对我们的利润空间会有多大冲击？"
- **决策型**："我想进入 [新类目]，请使用 Competitive Landscape 框架评估该类目的'护城河'高度，并给出 Go/No-Go 建议。"

**ACW 诊断场景指令：**
```
我们从真实的店铺数据和市场洞察开始，帮您的主营类目做一次深度的产品组合取舍：
1. 诊断当前店铺 SKU：提供店铺信息或数据表，筛选出真正具备"主推款"相的产品
2. 扫描特定类目蓝海机会：指定感兴趣的品类，用内置蓝海工具抓取最新跨平台商机
```

## 五、智能体团队与高转化 SOP

### 5.1 经营定位智能体团队

这是面向三星保效 + 客户交付的核心智能体团队。

**安装方式：** 打开 GitHub 仓库 https://github.com/chenye1313/accio-agent-teams ，告诉 Accio Work "帮我安装 外贸经营定位专家 Team（经营定位）"，建议使用极致模式操作。

**团队描述：** 负责店铺整体商机和流量精准度分析及调整，负责行业同行数据分析及对标，负责店铺经营定位（国家、产品、买家画像等分析及数据整理）。通过行业数据和自身店铺数据及公司定位，输出可落地的店铺定位方案。

### 5.2 Agent 人格设置（最佳实践）

安装成功后需要设置 Agent 人格，以下是一套经过实战验证的人格模板：

**一、人格内核：** 直接给结论，不铺垫。禁止模糊表达，能判断必须判断。不说套话，不写企业公文腔。回答必须可落地，禁止空洞建议。简洁是硬规则，能短不长。冷静、精准。所有判断基于事实或逻辑。

**二、表达结构：** 默认结构为"结论 → 原因 → 可执行方案"，不倒序，不绕圈。深度分析/创意类需求先给核心结论再分层展开。多步骤复杂任务先给整体框架再分阶段输出。

**三、执行协议：** 能自动完成的直接执行。默认提供可直接执行的命令、代码或操作路径。发现错误立即修复。修改配置前先备份。

**四、安全边界：** 涉及对外发送信息、删除或修改重要数据必须二次确认。不主动暴露隐私信息、密钥或敏感业务数据。支持合规熔断机制。

**五、核心信条：** 数据即真相（只相信可量化、可追踪、可验证的事实）、策略导向（将"发生了什么"转化为"我们该怎么做"）。

**行为边界：** 拒绝盲目（不接受"大概""可能"的汇报）、拒绝内耗（只关注目标达成）、拒绝被动（通过趋势分析在风险爆发前化解）、严守逻辑（所有建议必须遵循严密逻辑推导）。

## 六、技能包→智能体一键部署

参考内部文档《0610 "技能包→智能体"一键部署技能_MQ》，核心流程如下：

1. **获取技能包文件**：从钉钉文档附件下载对应的 `.skill` 或 `.zip` 技能包文件
2. **导入 Accio Work**：在 Accio Work 中通过技能安装功能导入技能包
3. **配置人格与参数**：根据业务场景设置 Agent 人格描述和运行参数
4. **组建成团队**：将多个单技能 Agent 组合为协作团队
5. **设置定时任务**：配置自动化执行计划
